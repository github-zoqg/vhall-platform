# 视频轮巡摄像头检测

用于 pc 观看端的摄像头设置

## 主要功能

检测及设置视频轮巡时使用的摄像头设备

## 依赖服务

1. mediaSettingServer

## 需要注意点

### 发送 diff

保存时发送的是 `diffOptions` ，即只发送变动的配置项，以让接受事件者知道哪些变动。
如果需要全量的 options 直接取 `mediaSetting.state` 即可

## 输入 api(向其他组件发送的指令)

| 方法名           | 参数 1      | 描述                            | 代码块 |
| ---------------- | ----------- | ------------------------------- | ------ |
| saveMediaSetting | diffOptions | 保存配置(传输变动的 options 项) |        |

## 输出 api(暴露给其他组件使用的方法)

| 方法名          | 方法说明           | 参数 | 描述 |
| --------------- | ------------------ | ---- | ---- |
| showCameraCheck | 展示摄像头设置窗口 |      |      |
