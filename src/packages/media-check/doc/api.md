# 媒体检查

用于 pc 发起端的媒体设置

## 主要功能

1. 按步骤检查 视频、录音、扬声器
2. 如果存在检测失败，则最终检测结果页报告失败明细并点击“重新检测”重来
3. 如果检测全部成功，则最终检测结果报告成功明细并点击“去直播”关闭媒体检查弹窗
4.

## 依赖服务

1. mediaCheckServer
2. roomBaseServer
