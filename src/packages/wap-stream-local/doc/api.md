## wap 本地流组件

用于 wap 本地推流相关逻辑<br>
包含：

- 创建本地流
- 发布本地流
- 退出全屏
- 静音上麦
- interactiveServer 事件监听
  - 上麦成功
  - 下麦成功
  - 麦克风/摄像头 设备状态变化
  - 同意上麦
- 异常错误处理集合：handleSpeakOnError

<br><br>
注：此组件由多人开发，由相关人员自行增加
