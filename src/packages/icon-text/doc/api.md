## 图标文字组合按钮

### 输入 api(向其他组件发送的指令)

```js
-------------------------------------
方法名： handleClick
描述：点击事件
参数：无
代码：window.$middleEventSdk?.event?.send(boxEventOpitons(this.cuid, 'handleClick'));
------------------------------------
```

### 输出 api(暴露给其他组件使用的方法)

```js
-------------------------------------
方法名： setSelectedState
描述：设置选中转态
参数：val，布尔类型 true-选中，false-不选中
------------------------------------
方法名： setDisableState
描述：设置禁用状态
参数：val，布尔类型 true-不可用，false-不可用
------------------------------------
方法名：setHiddenState
描述：设置展示隐藏状态
参数：val，布尔类型 true-展示，false-隐藏
------------------------------------
方法名：setText
描述：设置展示文本
参数：val，文本，字符串类型
------------------------------------
```
