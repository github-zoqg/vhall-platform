## 报名表单组件（PC）


1. 主要用于PC观看端
2. 题目类型固定，表单类型，收集用户信息
3. 根据入口的不同，表现形式也不同，作为报名表单的时候，自带dialog功能，方便使用。作为独立报名表单，则嵌入到独立报名表单模块里做模板使用

### 输出 api(暴露给其他组件使用的方法)

| 方法名 | 参数 | 描述 |
| ------ | ---- | ---- |
|     openModal   |  无    | 提供给pc观看端预约业使用，打开报名表单弹窗     |

### 依赖服务

- useRoomBaseServer
- useSignUpFormServer
