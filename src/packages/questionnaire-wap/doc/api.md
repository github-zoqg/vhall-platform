### wap 观看端问卷组件

#### 功能:

1. 提交问卷

#### 消息:

| 消息类型           | 描述     |
| ------------------ | -------- |
| QUESTIONNAIRE_PUSH | 推送问卷 |

#### 事件驱动相关方法:

| 方法名                   | 参数描述         | 描述                                    |
| ------------------------ | ---------------- | --------------------------------------- |
| open                     | 问卷 id          | 接收问卷按钮/聊天区的事件通信--打开问卷 |
| emitQuestionnaireVisible | 当前弹窗开关状态 | 派发事件通信通知播放器组件小屏显示      |

#### 问卷 SDK 事件:

| 事件类型 | 描述     |
| -------- | -------- |
| SUBMIT   | 问卷提交 |

#### 问卷 SDK 方法:

| 方法名称                | 描述     |
| ----------------------- | -------- |
| renderQuestionnaire4Wap | 展示问卷 |
