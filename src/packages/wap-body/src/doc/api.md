## wap 播放器及互动流父容器组件

包含功能如下：

- 直播结束占位图
- 播放器容器
- 互动流列表容器
- 桌面共享容器
- 插播容器
- 分组活动： 各种提示
- 小窗下的流/播放器容器拖动

<br><br>

### 子组件信息

Mask:

- 介绍：
  - 左右滑动页面提示
- 功能：
  - 当前活动只展示一次，存储 session
  - 距离头部高度随 head 是否显示而变

注：此组件由多人开发，由相关人员自行增加
