# 音量预览

**性质：通用组件 /**
用于 media-setting 和 media-check 预览音量

## 主要功能

1. 可视化预览音量大小，采集麦克风声音，声音越大，音量条越长，反之越短。

## 依赖服务

无

## 需要注意点

无依赖 server，为 chrome-api 原生实现
